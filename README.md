# Herve

Herve is a lightweight RISCV Instruction Set Simulator written in C.

The Herve name is a French person's name that happens to have similar pronunciation of "RV", shortcut for RISCV.

Herve has been tested for verilog-DPI integration into RTL testbenches using verilator as simulator.

Icarus verilog VPI integration has been tested as well.

Herve has been benchmarked against spike, and provides much faster execution times, due to its simpler processing.
